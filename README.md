🍸 Ashish | CV & Cocktail Portfolio Website

A simple personal website to showcase your CV and cocktail creations with photos and descriptions. This project is built using HTML, Tailwind CSS, and JavaScript (LocalStorage).


---

🚀 Features

📄 Display your CV / Professional Profile

🍹 Add and showcase cocktails with images & descriptions

💾 Data saved in browser local storage (no backend required)

📱 Fully responsive design using Tailwind CSS



---

📂 Project Structure

index.html   # Main website file


---

▶️ How to Use

1. Copy the code and save it as:

index.html


2. Open the file in your browser (Chrome recommended)


3. Navigate through sections:

CV → Your professional details

Cocktails → View your menu

Add New → Add new cocktail entries





---

➕ Adding a Cocktail

Fill in the form:

Cocktail Name

Image URL

Description


Click Add → It will instantly appear in your menu.
